UsuarioEntity

data class UsuarioEntity(val idUsuario: String, val nombre: String, val email: String, val fotoUrl: String? = null, val fechaRegistro: Long = System.currentTimeMillis())

Entidad Room que representa un usuario de la aplicación. Se almacena en la tabla USUARIO de la base de datos local.

Los datos del usuario se obtienen desde Firebase Auth y Firestore al iniciar sesión o registrarse, y se cachean en Room a través de dam.pmdm.tripplanner.data.repository.AuthRepository. Esto permite mostrar los datos del usuario sin conexión.
